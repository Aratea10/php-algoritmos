# PHP - Ejercicios con Algoritmos y formularios
Colección de ejercicios organizados por carpetas:
- `ejercicio-1`: Algoritmos 5.2, 5.3 y 5.4 (conversión cm→pulgadas/pies, función por tramos, raíces de ecuación cuadrática).
- `ejercicio-2`: Algoritmos 5.8, 5.9, 5.10 y dado con `rand()` (for, while, umbral S≥2, divisores, simulación de dado).
- `ejercicio-3`: Arrays numéricos y asociativos (horas extra, IBEX, rangos con signo, tabla de multiplicar).
- `ejercicio-4`: Formulario POST con selección múltiple y color, render de datos recibidos y muestra de color.
- `ejercicio-5`: Formulario POST con selección múltiple y color, render de datos recibidos y muestra de color.

## Requisitos
- PHP 8.x
- Servidor embebido de PHP o cualquier servidor web que ejecute PHP
  
Comprueba versión
```bash
php -v
```
